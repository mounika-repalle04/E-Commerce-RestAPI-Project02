# E-Commerce REST API Project (MVC + JSP)
## Project Description
This is a simple E-Commerce demo project developed using:
- Java
- Spring Boot
- Spring MVC
- REST API
- JSP
- HTML, CSS, JavaScript
- Apache Tomcat
The project allows users to add items using a browser-based UI without Postman.
## Features
- Add item with validation
- REST API integration
- MVC architecture
- Browser-based UI
- Input validations
- JSON response handling
## Technologies Used
- Java 17
- Spring Boot
- Spring MVC
- JSP
- REST Controller
- Apache Tomcat 11
- HTML, CSS, JavaScript
